Grids & Utilities

Flex Stack

Apply `bbc-stack` with a gap modifier to stack content vertically.

<div class="bbc-stack bbc-stack--gap-sm">…</div>

Row Utilities

Use `bbc-flex-row` (with optional `bbc-flex-row--wrap`) for inline meta information.

<div class="bbc-flex-row bbc-flex-row--gap-sm bbc-flex-row--wrap">…</div>

Responsive Grid

`bbc-grid` automatically adapts via CSS custom properties; add `bbc-grid--stack` for auto-fit columns.

<div class="bbc-grid bbc-grid--stack">…</div>
